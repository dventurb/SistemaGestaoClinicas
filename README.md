# Sistema de Gestão de Clínicas 
Aplicação para gestão de clientes, médicos e consultas de uma clínica, implementado em Linguagem C. O projeto utiliza conceitos como vetores estáticos, manipulação de ficheiros e estruturas, apontadores e programação modular, desenvolvido como parte da disciplina de Algoritmia e Programação.

## Introdução 
O Sistema de Gestão de Clínicas foi criado com o intuito de facil utilização e manipulação de dados em clínicas, permitindo realizar operações essenciais de gestão de clientes, médicos e consultas, oferecendo uma interface interativa com validações garantindo integridade nos dados.
